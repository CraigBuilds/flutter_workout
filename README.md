# Features

 - View Today's Workout, Historical Workouts, and Planned Workouts from the Home Page
 - Add Exercises to today's workout from the homepage
 - Add Sets to the currently selected Exercise in the set_logging_page
 - Quickly view all historical sets for a particular exercise from the History tab of the set_logging_page
 - NoSQL database integration using Hive
